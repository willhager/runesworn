## Runesworn

This is a (WIP) dice-based rogue-lite RPG video game built in the Godot engine. After Balatro came out in 2024, I found myself increasingly 
interested in video game development, and this is the result of that interest. Currently, loading this directory directly into Godot will allow you 
to run the game. I'm planning on releasing an early beta to some playtesters by August of 2026.
